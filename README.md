## Introduction

該網站是一個經典的貪食蛇遊戲，可以使用簡單版遊玩或是切換為困難版來進行遊玩，並且持續的挑戰，嘗試突破自己的最高紀錄!
### 遊玩方法 : 
鍵盤使用**方向鍵**或是**WASD**鍵可以操控方向，控制蛇要往上下左右任一方向行走，然後持續的讓蛇吃到黃色果實，但是必須得注意，不可以吃到自己的身體，否則遊戲就會結束。
